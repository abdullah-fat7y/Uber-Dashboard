# Uber-Dashboard
Developed an interactive Power BI dashboard to analyze sales trends, booking performance, revenue, and customer behavior. Designed dynamic KPIs, implemented DAX measures, and built drill-through analysis pages for deeper insights.

## Overview
This project presents an interactive Power BI dashboard designed to analyze sales trends, booking performance, revenue metrics, and operational efficiency.

The objective is to transform raw transactional data into actionable business insights that support data-driven decision-making.

This project reflects my engineering-oriented approach to data analysis: structured modeling, optimized calculations, and clarity in visualization.

---

## Business Problem

Organizations often struggle with:

- Monitoring overall revenue performance  
- Identifying lost bookings and revenue leakage  
- Comparing vehicle or service-type performance  
- Tracking operational distance efficiency  
- Making quick strategic decisions from fragmented reports  

This dashboard centralizes key metrics into a unified analytical view.

---

## Key KPIs

- **Revenue**
- **Completed Bookings**
- **Lost Bookings**
- **Average Distance**
- **Total Distance**

All KPIs dynamically respond to filters and slicers.

---

## Features

- Interactive KPI cards  
- Vehicle Type filtering  
- Drill-through analysis pages  
- Dynamic DAX measures  
- Clean, professional dashboard layout  
- Optimized data model  
- Data transformation using Power Query

## Technical Implementation

- Data cleaning using Power Query
